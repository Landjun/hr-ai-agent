工作城市：北京

4. 有完整的 AI 项目从 0 到 1 上线经验。

加分项（优先考虑）：
1. 熟悉 LangChain / Coze / Dify 等 Agent 开发框架。
2. 有提示词平台、知识库生产平台等实践成果。
3. 有开源贡献、论文产出或国家级竞赛荣誉。
4. 了解 Docker / K8s 部署与微服务架构。

面试重点：考察候选人的大模型工程落地能力、项目深度与复盘能力。
