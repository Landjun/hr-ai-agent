招聘岗位：网络安全渗透测试工程师
所属部门：安全攻防团队
工作城市：北京

岗位职责：
1. 负责对公司及客户的 Web、App、内网、业务系统开展渗透测试与安全评估。
2. 负责漏洞挖掘、复现与验证，输出高质量渗透测试报告并跟进修复。
3. 参与红队攻防演练（HVV/护网），完成内网渗透、提权与横向移动。
4. 编写安全工具与自动化脚本，提升攻防效率。

任职要求（必须满足）：
1. 本科及以上学历，3 年及以上渗透测试 / Web 安全经验。
2. 精通 OWASP Top10，熟悉 SQL 注入、XSS、SSRF、文件上传、反序列化等漏洞原理与利用。
3. 熟练使用 Burp Suite、Nmap、Metasploit、Cobalt Strike、sqlmap、Kali 等工具。
4. 掌握内网渗透、提权、免杀、横向移动等实战技能。
5. 熟悉 Python / Go，能独立编写安全工具与脚本。

加分项（优先考虑）：
1. 持有 OSCP / CISP-PTE 等证书。
2. 有 SRC 漏洞提交排名、CVE 致谢或 CTF 获奖经历。
3. 有代码审计或二进制安全能力。

面试重点：实战攻防能力、漏洞挖掘深度与渗透思路。
