招聘岗位：安全开发工程师
所属部门：安全工程团队
工作城市：北京

岗位职责：
1. 负责安全工具与安全平台（SAST/DAST、扫描器、安全中台）的设计与开发。
2. 负责将安全能力以 SDL 方式融入研发流程，推动安全左移。
3. 负责安全自动化建设，提升漏洞发现与收敛效率。
4. 以攻防视角设计与评审系统安全方案，参与威胁建模与代码审计。

任职要求（必须满足）：
1. 本科及以上学历，3 年及以上研发经验（安全方向优先）。
2. 精通 Java / Go / Python 之一，具备扎实的后端 / API / 微服务工程能力。
3. 熟悉常见漏洞原理与安全编码规范，理解攻防对抗。
4. 有安全工具 / 平台从 0 到 1 的开发经验。

加分项（优先考虑）：
1. 熟悉 SDL、等保合规、威胁建模。
2. 有开源安全工具、CTF 或安全证书。
3. 有代码审计经验。

面试重点：研发工程能力 + 安全攻防理解 + 平台建设经验。
