# 界面截图

本地运行 `streamlit run web/streamlit_app.py` 后，把截图保存到本目录，建议命名：

| 文件名 | 截图内容 |
| --- | --- |
| `dashboard.png` | 首页 Dashboard（Hero 横幅 + 指标卡 + 等级分布图） |
| `screening.png` | ③ 简历筛选：候选人排序表（总分进度条 + 等级徽章） |
| `interview.png` | ⑤ AI 模拟面试：一问一答 + 评分反馈 |

然后到根目录 [`README.md`](../../README.md) 的「界面预览」处取消对应图片注释即可展示。
